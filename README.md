# Chat-Buddy

It is a Full stack Chat Application using **Flutter** and **Firebase**. It allows **real-time** messaging to registered users. It uses Flutter Bloc for State Management.

### Sailent features:
- User Registration with validation
- Authentication
- Real-time messaging
- Notifications
- Search users
